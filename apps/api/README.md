# `apps/api`

NestJS backend for Coin Collector Companion. See root [CLAUDE.md](../../CLAUDE.md) and [docs/system-design.md](../../docs/system-design.md) for architecture and conventions.

All commands below run from the repo root.

```bash
pnpm --filter api start:dev   # dev server, watch mode
pnpm --filter api build       # production build
pnpm --filter api test        # unit tests
pnpm --filter api test:e2e    # e2e tests
```
