# `apps/web`

Next.js (App Router) frontend for Coin Collector Companion. See root [CLAUDE.md](../../CLAUDE.md) and [docs/system-design.md](../../docs/system-design.md) for architecture and conventions.

All commands below run from the repo root.

```bash
pnpm --filter web dev     # dev server
pnpm --filter web build   # production build
```
